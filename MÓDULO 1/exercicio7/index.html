Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag b (não semântica).

Nesta frase temos um destaque em negrito usando a tag strong (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag i (não semântica).

Nesta frase temos uma ênfase em itálitco usando a tag em (semântica).

Texto marcado

Podemos também criar um texto marcado usando a tag mark.

Texto grande e pequeno

Podemos também criar um texto grande e um texto pequeno usando as tags big e small.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, com a tag del.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag ins.

Existe também o sublinhado com a tag u (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3.

Texto subscrito

Para inserir coisas do tipo H2O.
